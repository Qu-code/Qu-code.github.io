Hexo搭建个人博客

发表于 2020-01-01 更新于 2020-01-22 分类于 Hexo博客 阅读次数：

Hexo是一个快速、简洁且高效的博客框架。利用Hexo框架可以快速搭建属于自己的博客，创造一片仅属于自己的网络空间。这篇博客介绍了windows操作系统如何使用Hexo创建博客，并利用Github让你的博客发布到互联网上。

准备工作

安装Hexo并不需要繁杂的操作和大型软件，只需安装下列应用程序即可

Node.js + Git

准备工作主要就是安装Node.js和Git。Node.js是一个基于Chrome V8引擎的Javascript运行环境，提供了一个让JavaScript运行在服务端的平台。Git是一个开源的分布式版本系统，可以有效、高速地处理从很小到非常大的项目版本管理。Git是为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件。

  • Node.js（Node.js 版本需不低于 8.10，建议使用 Node.js 10.0 及以上版本）
  • git

中国大陆用户用上述网址下载安装包可能会十分缓慢，我建议用下列淘宝镜像网址进行下载。

  • Node.js
  • git

安装过程

安装过程十分简单，选定好安装文件夹，一路next下去就可以了。不过在安装时需要注意以下两点。

  1. 安装Node.js的时候建议勾选Add to path，这样就避免了后续需要自己添加环境变量的麻烦。
  2. 安装Git时，注意勾选Git Bash Here

开始搭建博客

所有必备的应用程序安装完成后，就可以开始搭建个人博客了。首先建立一个空文件夹，打开文件夹点击右键选择Git Bash Here打开命令行

在命令行中输入如下指令安装hexo博客
$ npm install -g hexo-cli
经过一段时间的下载和安装，hexo博客就成功安装了。

需要注意的是，中国区的用户使用npm可能速度会很慢，我们可以首先下载淘宝npm镜像
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
安装好之后，便可以使用cnpm安装Hexo。由于本文挑战发起者位于加拿大，可以忽略这里，使用npm指令。
$ cnpm install -g hexo-cli

安装完成后便可以正式开始建立自己的博客，在命令行中输入如下指令：
hexo init
这时你就会发现刚才所创建的空文件夹中多出了一些文件夹和文件。具体每个文件夹以及文件对应什么功能，我们在这里就不作深入讨论了。我可以告诉你的时所有博客的内容都将会包含在这个文件夹中，当你经过一段时间的捣鼓反而造成了不可恢复的错误时，不要惊慌，删除掉这个文件夹重新来过即可。

你可以输入
hexo -v
查看hexo 博客的版本

生成静态页面
hexo g
启动博客
hexo s

这样一个博客就搭建完成了，你可以打开浏览器，输入网址localhost:4000来浏览你的网页。你所创建的初始网页讲如下图所示，并且还会有一篇hexo所写的博文。

将博客部署到远端

这样搭建的博客只能在本地电脑上显示，这样显然是不够的。要想让自己的博客被他人浏览还需要将博客部署到远端服务器上。对于家境贫寒的人来说，每个月支付一笔钱来维持服务器或是在家里自己搭建一台服务器主机都是不现实的。我也是个家境贫寒的人，只能通过免费的方式部署博客。免费部署的方法有多种，下面我仅介绍将博客部署到GitHub上的方式。Github是一个面向开源及私有软件项目的托管平台。该平台已经成为世界各地的人互相交流、学习的大型平台，相信任何程序员都会有在这个平台上学习交流的精经历。我们所使用的是Github所提供的GitHub Page服务，它是免费的静态网页托管服务，可以用来托管博客、项目官网等静态网页。这无疑是我们家境贫寒之人的福音。话不多说，我们开始把！

配置SSH Key

要使用git工具首先要配置SSH Key。

安装远程部署插件

在命令行中输入
npm install hexo-deployer-git --save
